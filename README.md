# Projeto-de-Sistema-Embarcado---SIIV
Projeto de estacionamento inteligente que detecta vagas disponíveis em tempo real. O sistema identifica espaços livres e ocupados, ajudando a organizar o fluxo de veículos e reduzir o tempo de busca por vagas.

# Sistema Embarcado - SIIV

Projeto desenvolvido para controle inteligente de vagas em estacionamento utilizando sensores.

## Funcionalidades
- Detecção de vagas
- Monitoramento em tempo real
- Interface visual
